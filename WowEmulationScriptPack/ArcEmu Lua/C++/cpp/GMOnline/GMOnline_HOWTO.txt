### Hungarian version ###

Hogy használjuk a szkriptet:

1. Futtassuk le a GMOnline.patch filet az ArcEmu forráskódunkon.
1b. Nyissuk meg Visual studioval az arcEmut és az 'ExtraScripts' projekt Scripts mappáján jobb klikk->add->Existing item. A filet a /scripts/src/ExtraScripts/GMOnline_HUN.cpp neven talalod.
2. Forditsuk újra az ArcScripteket.
3. Kész :D
4. (info) Az ArcEmu-optional.conf ban lehet állitani hogy be legyen kapcsolva vagy nem. (ki: GMOnline="0", be: GMOnline="1")

### English version ###

How to use this script:

1. Apply the GMOnline.patch file on your ArcEmu sourcecode.
1b. Open arcemu with VS2008 and in 'ExtraScripts' project's Script folder do rightclick->Add->Existing item. The file is found in /scripts/src/ExtraScripts/GMOnline.cpp
2. Re-compile ArcScripts.
3. Finish :D
4. (info) In arcemu-optional.conf you can turn this script on&off. (off: GMOnline="0", on: GMOnline="1")
